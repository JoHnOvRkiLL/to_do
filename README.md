# To-Do List App

This is my first React Native application, a simple and intuitive To-Do List app that helps users organize and manage their daily tasks efficiently. 

## Features

- **Add Tasks**: Quickly add tasks to your to-do list.
- **Mark as Complete**: Mark tasks as complete with a single tap.
- **Responsive Design**: Works seamlessly on both Android and iOS devices.

## Technologies Used

- **Framework**: React Native
- **State Management**: React's useState Hook
- **Styling**: React Native Stylesheet

## Installation

Follow these steps to run the application locally:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/JoHnOvRkiLL/to_do.git
   cd todo-app
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start the Development Server**:
   ```bash
   npx expo start
   ```
